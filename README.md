# شهربازی عددهای اعشاری

این پوشه برای آپلود در GitHub Pages و سپس ساخت APK با PWABuilder آماده شده است.

## فایل‌ها
- index.html: فایل اصلی بازی
- manifest.json: فایل معرفی اپلیکیشن
- sw.js: فایل سرویس‌ورکر برای نصب و اجرای PWA
- icon-192.png و icon-512.png: آیکون‌های اپلیکیشن

## روش استفاده
1. یک Repository در GitHub بسازید.
2. همه فایل‌های داخل این پوشه را در ریشه Repository آپلود کنید.
3. از Settings > Pages، گزینه Deploy from branch را فعال کنید.
4. لینک GitHub Pages را در سایت PWABuilder وارد کنید.
